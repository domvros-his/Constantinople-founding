How to deploy on GitHub Pages

1) Upload ALL files/folders in this package to the folder GitHub Pages serves (repo root or /docs).
2) Make sure these paths are reachable:
   - index.html (this wrapper)
   - app.html (original Animate export)
   - index.js
   - blibs/1.0.0/createjs.min.js
   - bimages/_preloader.gif
   - credits/info.html

If you open locally, use a local server (VS Code Live Server or `python -m http.server`).
